# puukkopaja

Single-page site for a Finnish knife workshop. Hosted on GitHub Pages — no build step, open `index.html` in a browser.

## Stack

HTML + Tailwind CSS (CDN) + vanilla JS. Font Awesome, Google Fonts (Inter, Lora).

## Deploy

1. Push to GitHub
2. **Settings → Pages** → Branch `main`, `/ (root)`
3. Save

## Structure

```
index.html     Landing page
main.js        Interactions (menu, scroll, form)
style.css      Custom styles (animations, hero, toasts)
public/        Images
```
